# This function writes citation data to disk in different formats

This function writes citation data to disk in different formats

## Usage

``` r
write_citations(citations, type = c("ris", "txt", "csv", "bib"), filename)
```

## Arguments

- citations:

  A data frame containing citations - usually post-deduplication

- type:

  export type

- filename:

  output file name

## Value

file export
